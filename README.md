#  To-Do List App

A clean and responsive To-Do List app built using HTML, CSS, and JavaScript.
Tasks are saved in localStorage so they stay even after refreshing the page.

##  Live Demo
Open index.html in any browser to run locally

##  Features
- Add new tasks instantly
- Mark tasks as complete with one click
- Delete individual tasks
- Filter tasks by All / Active / Completed
- Tasks saved in browser localStorage
- Responsive glassmorphism dark UI
- Press Enter to quickly add tasks

##  Tech Stack
- HTML5
- CSS3 (Glassmorphism design)
- JavaScript (ES6+)
- localStorage API
